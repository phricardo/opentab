# OpenTab

OpenTab is a lightweight browser start page for faster search, favorite websites, and useful shortcuts. On Microsoft Edge, it gives you back control of the new tab page.

## Why OpenTab exists

Microsoft Edge is a good browser, but its default new tab page is overloaded with distractions, ads, news, widgets, and limited customization.
OpenTab was created to debloat the new tab experience and give users a cleaner, faster, and more personal start page.
The goal is simple: when you open a new tab, you should see what matters to you.

## Features

- Clean and minimal browser start page
- Fast search from a focused interface
- Favorite websites always visible
- Useful web shortcuts in one place
- Firefox-inspired visual experience
- Customizable options page
- Native English and Brazilian Portuguese support
- Open source forever

## Built for Microsoft Edge

OpenTab is especially useful on Microsoft Edge because it helps you replace the default new tab page with your own start page.

Instead of accepting the default Edge experience, you can use OpenTab to make the new tab page cleaner, faster, and more aligned with how you browse.

## Inspired by Firefox

The visual experience of OpenTab is inspired by the simplicity of Firefox’s new tab page: clean spacing, useful shortcuts, and a start page that feels calm instead of noisy.

OpenTab keeps that idea, but makes it customizable and open source.

## Customization

OpenTab can be customized from the options page.

You can adapt the experience to match your personal style, your company identity, or any visual direction you prefer.

Possible customizations include:

- Brand identity
- Favorite shortcuts
- Visual preferences
- Search experience
- Language preferences

This makes OpenTab useful both for personal browsing and for companies that want a cleaner internal browser start page.

## Languages

OpenTab supports:

- English
- Brazilian Portuguese

The interface is designed to feel natural in both languages.

## Open source forever

OpenTab is open source and will remain open source forever. You are free to use it, modify it, adapt it, and contribute to it.
